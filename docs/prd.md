### Webpage for the artis Diaz

### Instructions:
This is a landing page for a famous rapper named Diaz. It is a simple one pager with a few simple image effects, mainly the hero section (described under Herosection in this document)

Important! This is a responsive site - On mobile the menu will be a hamburger menu that when clicked, lists all the menu items in a nice manner. Use microanimations to gracefully display the menu.



### Menu

The menu consists of 8 seperate menu items - All menu items are anchor links to a section in the same page. each link represents a section - even if there is little content for each section, give the section some room (head space).

The menu items are:

- Musikk
- Bio
- Bilder
- Booking
- SoMe
- Foredrag om fluefiske
- Fluefiskefilmer
- Kontakt

Visual treatment: White text, Google fnt: Noto Serif 400
The menu is positioned fixed in the top of the page and will not scroll away.

### Hero
The hero section hase a full screen background image. The image is in fact TWO images, place exactly on top of each other. Full dvh. When the user scrolls down, the top image will slowly fade away, revealing the bottom image, when the user scrolls back the iage reappears (fade). 

The two mages are located in the media folder called hero-a.jpg and hero-b.jpg\
Stacking : bottom image is hero-b.jpg, top image is hero-a.jpg. The image that fades away with the scrolling is the top image.
